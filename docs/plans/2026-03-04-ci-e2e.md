# CI E2E Playwright Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add an `e2e` job to `.github/workflows/ci.yml` that starts the FastAPI server and runs the Playwright E2E suite on every push, blocking merges on failure.

**Architecture:** One new job in the existing `ci.yml` — no new workflow file. The job installs Playwright + Chromium, starts uvicorn in the background, waits for the health endpoint to respond, then runs `scripts/e2e_full_ui.py`. Screenshots are uploaded as artifacts when the job fails. `playwright>=1.40.0` is added to `requirements-dev.txt` so both CI and local devs use the same version.

**Tech Stack:** GitHub Actions, uvicorn, Playwright (Python sync API), httpx (already in requirements-dev.txt).

---

## Background

Key locations:
- **CI workflow:** `.github/workflows/ci.yml` — has three jobs: `test-and-docs`, `integration-tests`, `reconcile-check`
- **E2E suite:** `scripts/e2e_full_ui.py` — exits non-zero if any step FAILs; writes screenshots to `screenshots/e2e-full-{date}/`
- **Health endpoint:** `GET /api/v1/system/health` — returns 200 when server is ready
- **Server command:** `uvicorn src.api.main:app --host 127.0.0.1 --port 8000`
- **requirements-dev.txt** — currently has `ipython`, `jupyter`, `sphinx`, `httpx`; does NOT have `playwright`

The E2E suite has zero Oracle/DB dependency — all four workflows (quick_test, recent_runs, mapping_generator, api_tester) drive only the UI. The server starts without Oracle env vars.

### Step 2: Add the `e2e` job to `ci.yml`

In `.github/workflows/ci.yml`, append the following job after the `reconcile-check` job (after line 82):

```yaml
  e2e:
    runs-on: ubuntu-latest
    needs: test-and-docs

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements-dev.txt
          pip install -e .

      - name: Install Playwright browser
        run: playwright install chromium --with-deps

      - name: Start server
        run: |
          uvicorn src.api.main:app --host 127.0.0.1 --port 8000 &
          python - <<'EOF'
          import time, httpx, sys
          for i in range(20):
              try:
                  r = httpx.get("http://127.0.0.1:8000/api/v1/system/health", timeout=2)
                  if r.is_success:
                      print("Server ready")
                      sys.exit(0)
              except Exception:
                  pass
              time.sleep(0.5)
          print("Server did not start in time")
          sys.exit(1)
          EOF

      - name: Run E2E suite
        run: python3 scripts/e2e_full_ui.py

      - name: Upload screenshots on failure
        if: failure()
        uses: actions/upload-artifact@v4
        with:
          name: e2e-screenshots
          path: screenshots/e2e-full-*/
          retention-days: 7
```
